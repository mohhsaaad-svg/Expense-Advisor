## 30-minute monthly ritual

| Minutes | Task | What to look for |
|---|---|---|
| 0–5 | Check Google Alert digest | Press, blog posts, product announcements since last month |
| 5–10 | Open App Store listings | Version updates; read the 3 most recent reviews for new complaint patterns |
| 10–15 | Open changelog/blog | Feature launches — note category (pricing, UX, integrations, AI) |
| 15–20 | Scan jobs pages | 3+ ML/AI roles = AI feature coming; Android roles at Copilot = their moat closing |
| 20–25 | Check pricing pages | Tier changes, price increases, new plan names (compare Wayback Machine if unsure) |
| 25–30 | Update this file | Add a change-log row; flag anything that warrants a response |

**Ember-specific tripwires** (escalate immediately if seen):
- Copilot ships Android or web → our platform advantage shrinks; accelerate habit/advisor differentiation
- YNAB or Monarch adds streaks/gamification → habit angle contested; double down on advisor + warmth
- Cleo fixes its trust problem (rating recovers above 4.0) → AI-coach lane gets a funded rival
- Origin ships a daily-spending coach → watch their framing vs. regulated advice
